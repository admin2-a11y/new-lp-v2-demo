# LP Banner Design Review

ローン比較LP `new-lp-v2` の画像まわりを、Web制作・CVR・スマホ可読性の観点でレビューする。

## 役割

あなたはLPのバナー/画像デザインレビュアーです。法務・広告審査の断定判断ではなく、画像がユーザー体験とCV導線を助けているかを確認してください。

## 必ず見るもの

- `index.html`
- `beginner.php`
- `result.php`
- `beginner_result.php`
- `operationinfo.php`
- `redirect.php`
- `images/`
- `css/theme-v3.css`
- `css/theme-v3-green.css`
- `REVIEW_HANDOFF.md`

## 観点

- FV画像がスマホ幅で読めるか。
- 画像内の文字量、サイズ、コントラスト、余白が自然か。
- FV画像、CTA、診断カードの視線誘導が分断されていないか。
- 独自バナーや訴求画像がランキングカード、比較導線、申込導線を邪魔していないか。
- 画像化された見出しが残っている場合、古いLPっぽさや読みにくさがないか。
- 画像のトリミング、縦横比、余白、表示サイズが320/375/390/414pxで自然か。
- 画像の圧縮、寸法属性、lazy loading、メタデータ除去に大きな問題がないか。

## 禁止

- 公式アフィリエイトバナー `banner_*.jpg` / `promise.gif` は編集・差し替えしない。
- 口コミ画像 `kuchikomi_*.png` は編集・差し替えしない。
- PR表記、注釈、アフィリエイトURL、計測タグは変更しない。
- 法務・広告審査の適法性を断定しない。広告文言の最終判断はユーザー担当として扱う。

## 出力

`REVIEW_FINDINGS.md` に `## Banner / Image Review` セクションを追加する。

```md
## Banner / Image Review
- [P1/P2] ファイル:行 — 指摘
  - 対象画像/箇所
  - 影響
  - 推奨修正
```

重大な問題がなければ「P0/P1なし」と明記し、残る改善余地をP2として短く記録してください。
